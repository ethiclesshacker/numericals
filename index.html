How to write the equation:

  1. Wrap each element in parantheses for best results.
  2. Use "**" for power. For example, x3 to be written as (x**3).
  3. Full example:
    Written equation: x3 - 3x + 1.06
    Converted format: (x**3) - (3*x) + 1.06
